# 🌍 Wanderlust – Travel Listing Platform

A full-stack web application that allows users to explore, create, and manage travel listings with interactive maps and dynamic filtering.

---

## 🚀 Live Demo
🔗 https://nestify-zjle.onrender.com

---

## ✨ Features

- 🔐 User Authentication (Signup/Login)
- 🏠 Create, Edit, Delete Listings
- 🗺️ Interactive Map Integration
- 🔎 Location-based Search
- 🧩 Category Filters (Beach, Mountain, City, etc.)
- 💰 Dynamic Pricing with GST Toggle
- ⭐ Reviews and Ratings System
- 📱 Responsive UI Design

---

## 🛠️ Tech Stack

- **Frontend:** EJS, Bootstrap, CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Maps:** MapLibre
- **Authentication:** Passport.js
- **Deployment:** Render
